# Commands phone.system.rish refuses to run.
#
# One REGEX per line, matched with re.search against the raw (un-lowercased)
# command; blank lines and #-comments ignored. Regex, not fnmatch: the
# literal string "rm -rf /" never matches "rm -rf / --no-preserve-root" (D10).
#
# This is a safety net, not a security boundary. It stops accidents, not
# attacks — anyone who can edit this file can empty it. Installed to
# ~/.config/phone-agent/rish_blocklist.txt on first use; edits there are
# picked up on the next call without restarting the service.

# File system destruction
^rm\s+(-[a-zA-Z-]+\s+)*/(\s|$|\*)

# Partition manipulation
^mkfs\.
^fdisk
^dd\s+.*if=/dev/(zero|u?random)

# System modification
mount\s+-o\s+remount,rw\s+/

# Factory reset
wipe_data
recovery\s+--wipe_data
^fastboot

# Bootloader
flash_image
unlock_bootloader
